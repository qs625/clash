# AUR Templates

- `myclash/` is the active AUR template for this repository.
- `myclash/` is kept in sync with successful `v*` GitHub release builds.
- New AUR updates should be based on `myclash/`.
- `webkit2gtk-4.1` is an official Arch Linux package from `extra`, so the AUR recipe already depends on the system WebKitGTK package rather than an AUR-only package.

All older local AUR template copies have been removed.
